# 北斗短报文野外救援指挥系统 (BDS Rescue System)

本系统是一款基于 **BDS-3 (北斗三号)** 短报文通讯协议模拟的野外救援指挥平台。系统集成了实时地图监控、手持终端仿真、双向通讯链路及后端数据审计功能，旨在模拟极区、沙漠等无地面网络覆盖环境下的紧急救援协同流程。

## 🛰️ 核心功能

- **战术指挥台 (Command Dashboard)**:
	- 集成 **Baidu Map WebGL 1.0** 引擎，支持 3D 视角与倾斜渲染。
	- 实时标注 SOS 报警点（红色）与救援分队（蓝色）位置。
	- 动态遥测数据流：经纬度、海拔、移动速度及卫星锁定状态实时监控。
- **手持终端仿真 (Terminal Simulator)**:
	- 模拟北斗三号手持机 UI 界面。
	- 支持一键 SOS 发射、位置自动附着上报及短报文文本交互。
	- 模拟卫星链路 1.2s 传输延迟，还原真实通讯感。
- **数据管理中心 (Data Management)**:
	- **报文审计**: 全量记录上行与下行报文，支持关键词检索。
	- **资源台账**: 实时统计救援队伍编制、状态及携带装备。
	- **系统统计**: 可视化展示救援响应率、信道负载等核心指标。
- **安全准入 (Auth Logic)**:
	- 伪登录/注册模块，支持自定义操作员编号接入系统。

## 🛠️ 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **地图引擎**: Baidu Map JavaScript API GL v1.0
- **样式方案**: Tailwind CSS (移动优先、自适应设计)
- **图标库**: Lucide React
- **动效**: Tailwind Animate / CSS Keyframes

## 🚀 本地部署步骤

### 1. 环境准备

- 确保已安装 **Node.js** (推荐 v18.0 或以上版本)。
- 获取百度地图浏览器端 AK（系统已默认配置测试 AK）。

### 2. 项目初始化

创建一个新目录并保存相关文件：

```
mkdir bds-rescue-system
cd bds-rescue-system
```

### 3. 安装依赖

在根目录下运行以下命令安装核心依赖：

```
npm install vite @vitejs/plugin-react react react-dom lucide-react
```

### 4. 启动开发服务器

运行以下命令启动项目：

```
npx vite
```

### 5. 访问地址

启动成功后，在浏览器访问：

http://localhost:5173

## 📂 项目结构建议

```
bds-rescue-system/
├── index.html          # 系统入口
├── package.json        # 依赖与脚本配置
├── App.jsx             # 核心逻辑 (包含指挥台、终端、数据管理所有模块)
└── README.md           # 项目部署说明文档
```

## ⚠️ 注意事项

1. **地图加载失败**: 若地图显示“Script Error”或白屏，请检查百度地图 AK 是否开启了“白名单限制”。测试时建议在百度地图控制台将 AK 的域名白名单设置为 `*`。
2. **浏览器兼容性**: 必须使用支持 WebGL 的浏览器（推荐 Chrome, Edge, Safari 最新版）。
3. **坐标偏置**: 系统采用 BD-09 坐标系，若接入其他 GPS 数据请注意进行坐标转换。
